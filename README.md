# 🏨 Hotel Santos Beachfront  

A modern and responsive hotel booking **UI prototype**, designed as part of a university project. This front-end project showcases the visual design of a beachfront hotel website, built with **HTML, CSS, and JavaScript**.  

## 🌟 Features  
✅ Fully responsive design for mobile and desktop  
✅ Interactive booking form (front-end only)  
✅ Modern UI with smooth animations  
✅ Styled with **CSS Flexbox & Grid**  
✅ JavaScript for basic interactivity  

## 🛠️ Technologies Used  
- **HTML5** – Structuring the website  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla)** – Interactivity
